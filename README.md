# ZabbixInstaller With OpenSSL

### Installation
```shell
git clone https://github.com/AungThuMyint/ZabbixInstaller.git
cd ZabbixInstaller/
chmod 777 ZabbixInstallerOpenSSL.sh
./ZabbixInstallerOpenSSL.sh
```
